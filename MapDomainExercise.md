# Atividade: Mapeamento de domínio

## Entidades
- Produto
- Fornecedor
- Estoque
- Pedidos (compra/venda)
- Alerta

## Casos de uso
- Registrar um produto
- Buscar produto(s)
- Rastrear um produto
- Enviar alerta por email
- Enviar alerta por notificação
- Buscar alerta(s)
- Criar pedido de venda
- Criar pedido de compra
- Buscar pedido
